ChartIQ Angular Application

The chartiq-angular-app project is a toolkit of Angular components. The following components create complete charting applications: (Select the component names to see the applications.)

  • AdvancedChartComponent

    Creates a chart with a full-featured user interface. AdvancedChartComponent is the equivalent of ChartIQ's technical-analysis-chart.html advanced template.

  • MultiChartComponent

    Displays two advanced chart components side by side in the same document.

  • ActiveTraderComponent

    (To enable this application, uncomment the Active Trader route in app.module.ts and enable the ActiveTraderComponent link in route-list.component.html.)

    Features the advanced chart component enhanced with the following plug-ins:

    • Trade From Chart
    • Order Book
    • Trade Book
    • Market Depth Chart

  • TermStructureComponent

    (To enable this application, uncomment the Term Structure route in app.module.ts and enable the TermStructureComponent link in route-list.component.html.)

    Creates a term structure chart for working with non–time series data.

  • CustomChartComponent

    Integrates native Angular components with ChartIQ web components.

  • HelloWorldComponent

    Creates a basic chart with no user interface as a starting point for using the ChartIQ API in Angular.
